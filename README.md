# DontFall 🎮

A vertical platformer built in Unity with combo-based scoring and platform types  
(Wood, Ice, Fire, Moving). Includes a Streamlit web dashboard for viewing runs and stats.

## 🎮 Game Features
- Combo system with score multipliers  
- Randomized platform types with rarity  
- Moving, icy, fire, and breakable platforms  
- High score saving  
- Juice: score pop + combo UI
